# sama-energy

Lean `pnpm`/Turbo monorepo scaffold with a Next.js frontend and an intentionally empty backend placeholder.

## Apps

- `apps/web`: Next.js 16 App Router app using TypeScript and Material UI
- `apps/api`: placeholder backend folder for future work

## Commands

- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm lint`
